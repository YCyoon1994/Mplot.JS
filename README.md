
# SNU material data center

## Introduction

Plot the lots of Simple Material charts
using pury javascript, css, html Canvas.
Easy install and simple utilize.

* Density of States [Demo]
* Band structures [Demo]
* Phase diagram [Demo]
* Lattice Structure [Demo]
